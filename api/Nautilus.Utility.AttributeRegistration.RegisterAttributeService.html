AddTypedDependencyInjector<T>(IDependencyArgumentInjector)

Adds a dependency injector for a given type. Types can be associated in two different ways: the argument's type and the argument's attributes. First argument attributes are checked, an example of this is the AssetLoadAttribute which takes precedence regardless of the argument type. Next argument types are checked, examples include PrefabInfo or TechType for their respective injectors.
public void AddTypedDependencyInjector<T>(IDependencyArgumentInjector injector)
