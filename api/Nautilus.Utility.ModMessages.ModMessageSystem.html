SendGlobal(string, params object[])

Sends a global message to every ModInbox that exists, and even to ones that will exist in the future. If a message is not read immediately by any inbox, it will be held until read.

IMPORTANT: Global messages can NOT be held after patch time has completed (once you have left the main menu).
